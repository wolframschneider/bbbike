Supported output formats are: [more help]
  • Protocolbuffer Binary Format (PBF) format
  • OSM XML, compressed with gzip, bzip2 or 7-zip (xz)
  • o5m, compressed with gzip or bzip2
  • Garmin format, with styles for OSM, cycle map, BBBike and leisure (freizeitkarte). Routing is enabled
  • Esri shapefile format
  • Osmand (OBF) format for Android devices
  • mapsforge format for Android devices
  • Navit format for Linux, Windows, Android, OpenMoko Freerunner, Wince, Nokia n800 Internet tables, iPhone, Zaurus
  • libosmium OPL format
  • SRTM Contours for Garmin, Osmand, and PBF
