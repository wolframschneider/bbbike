Want a polygon instead a box? Drag a box on the map, and then select a polygon on the map.

Choose Manually select a different area (if you haven't already). A message "Drag a box on the map to select an area" shows up. Click and hold the left mouse button, draw the bounding box, release the left mouse button. Choose add points to polygon. Click on the box on the map and add points. You can add as much points as you want, resize or move the polygon:
