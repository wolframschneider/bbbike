Hilfe
Unterstützte Ausgabeformate sind:
  • Protocolbuffer Binary Format (PBF) Format
  • OSM XML, komprimiert mit gzip, bzip2 oder 7-zip (xz)
  • o5m, komprimiert mit gzip or bzip2
  • Garmin Format, mit Styles für OSM, Fahrradkarte und leisure (freizeitkarte). Routing ist aktiviert
  • Esri shapefile Format
  • Osmand (OBF) Format für Android Gerärte
  • mapsforge Format für Android Gerärte
  • Navit Format für Linux, Windows, Android, OpenMoko Freerunner, Wince, Nokia n800 Internet tables, iPhone, Zaurus
