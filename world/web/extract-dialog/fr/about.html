extraits BBBike

vous permet de zones extraits de Planet.osm dans OSM , PBF , o5m, , Navit , Osmand , mapsforge ou ESRI Shapefile format. La taille de la zone maximale est de 24,000,000 km carré, soit la taille du fichier 768 Mo. Il faut compter entre 15-30 minutes pour extraire une zone. Vous serez averti par e-mail si votre extrait est prêt à télécharger. Pour plus d'informations, s'il vous plaît lire le page d'aide.

pris en charge formats de sortie sont les suivants:
  • Protocolbuffer (PBF) format
  • OSM XML, compressé avec gzip, bzip2 ou 7-zip (xz)
  • o5m, compressée avec gzip ou bzip2
  • Garmin format, avec des styles pour OSM, et de loisirs ( freizeitkarte). Le routage est activé
  • ESRI Shapefile format
  • Osmand (OBF) format pour les appareils Android
  • mapsforge (carte) format pour les appareils Android
  • Navit format pour Linux, Windows, Android, OpenMoko Freerunner, WinCE, tables Internet Nokia N800, iPhone, Zaurus

Le champ e-mail est requise, vous en serez informé par e-mail si votre extrait est prêt pour le téléchargement. S'il vous plaît nous meaningfull nom pour l'extrait. Les valeurs valides pour la longitude -180 sont .. 180, et pour la latitude -90 .. 90.
