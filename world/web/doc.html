BBBike @ World - Documentation

BBBike is a route planner for cyclists in Berlin. It is now ported to other cities around the world - thanks to the OpenStreetMap project!

There are 2 versions of BBBike, a web-based and a standalone.

Standalone Application

Get the full power of BBBike and download the standalone perl/tk version!

Map Data

BBBike @ World use map data by the OpenStreetMap Project. The database for BBBike @ World will be updated bi-weekly. Please visit the OpenStreetMap Wiki how to participate in the project and update the map yourself!

Supported Browsers

Any modern browser which conforms to the W3C standards:
  • Firefox 3.x, 4.x
  • Safari 4.x, 5.x, Google Chrome
  • Opera 9, 10
  • IE8.
JavaScript is required for Google/OpenStreetMap map features or the street name suggestion service. Cookies are optional to save your preferences.

Source Code

The BBBike source code is available at http://github.com/eserte/bbbike

Known Problems

  • Routing: some streets are not connected at bridges
  • Berlin: many cool features are only available for Berlin
  • There is no App for mobile devices as the iPhone or Androids. A design study is available.
