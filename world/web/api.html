BBBike @ World - API Documentation

BBBike is a route planner for cyclists in Berlin. It is now ported to other cities around the world - thanks to the OpenStreetMap project!

XML API in a Nutshell

BBBike has a simple XML API. You can get the routing results in the formats GPS routes, GPS tracks and Google Earth (KML). To get XML result output instead HTML, just replace the URL parameter output_as with one of the values:

Nameparametersdescription
GPS Routeoutput_as=gpx-routeGPX route with waypoints for GPS navigation, up to 256 points
GPS Trackoutput_as=gpx-trackGPX with up to 1024 points, no navigation
Google Earth (KML)output_as=kml-trackview route with Google Earth
Examples
/api/0.2/bbbike/?startc=13.377846%2C52.516265;startname=Brandenburger%20Tor;zielc=13.404294%2C52.535019;zielname=Zionskirche;pref_seen=1;pref_speed=20;pref_cat=;pref_quality=;pref_specialvehicle=;scope=;output_as=kml-track

JSON API in a Nutshell

The JSON APIs for street names or Point of Interests (POI) suggestions works as on the web site.

Examples

Searching for street names: /api/0.2/cgi/api.cgi?namespace=dbac;city=bbbike;query=wash
Plotting street names or POI: /api/0.2/cgi/street-coord.cgi?namespace=3;city=bbbike;query=washing
Reverse geocode: /api/0.2/bbbike/?api=revgeocode;lon=13.352005;lat=52.538919
